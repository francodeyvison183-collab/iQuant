# ADR-0004 微信小程序优先

## 背景

iQuant 面向普通用户，产品需要低门槛、易触达，并适合碎片化使用。

微信小程序具备较低使用门槛、社交生态、订阅消息和微信登录能力，适合作为 MVP 用户端。

## 决策

MVP 用户端优先做微信小程序。

服务端和核心算法保持平台无关：

- 策略生成。
- 回测。
- 盲测回放状态管理。
- 执行偏差诊断。
- AI 辅助分析。

这些能力通过 API 提供给小程序端。

## 影响

- 项目应用目录使用 `apps/miniprogram` 作为用户端入口。
- 小程序端只负责交互和轻量状态管理。
- K 线图、标注和盲测交互需要优先验证移动端触控体验。
- 后续如需 Web 或 App，可复用后端服务和领域包。
