# 02 行为策略生成模块

> 文件名保留 `02-ideal-strategy-generation.md` 以兼容历史链接；产品语义为 **行为策略（Behavioral Strategy）**，非开卷「理想策略」优先。

版本：V0.2（方案 A）  
关联：[ADR-0011](../../decisions/ADR-0011-blind-replay-primary-strategy-path.md)

## 1. 模块定位

行为策略生成模块根据**多轮盲测回放**样本（及一致性达标结论），生成可解释、可回测、可版本化的候选策略 DSL。

**主输入**：`blind_replay` 操作序列 + 每 bar 特征快照。  
**禁止**默认将 `historical_labeling` / `label_*` 作为主策略输入（可作 P2 对照）。

## 2. 用户价值

- 将用户在训练中表现出的买卖习惯，转化为明确规则。
- 用户确认规则后再回测，避免黑盒「系统替你做主」。
- 为优化建议（参数/规则）提供结构化载体。

## 3. 核心边界

### 3.1 本模块负责

- 读取 blind 样本与一致性报告。
- 在限定模板内归纳规则（突破、均线、持仓周期等）。
- 生成策略 DSL 与自然语言规则列表。
- 候选策略排序与版本字段。

### 3.2 本模块不负责

- 盲测交互。
- 一致性评分（前置用例）。
- 回测执行。
- AI 直接输出买卖信号。

## 4. 生成方式（MVP）

```text
多轮 blind_action + features_snapshot
  -> 一致性达阈值？
       否：不生成 DSL，返回训练建议
       是：模板匹配 + 参数拟合
  -> 候选 DSL（1–3）
  -> 用户确认
  -> 交付回测模块
```

暂不采用不可解释的黑盒模型作为主路径。

## 5. 内置策略模板（示例）

| 模板 | 买入逻辑 | 卖出逻辑 |
| --- | --- | --- |
| 均线突破 | 收盘价上穿 MA20 | 跌破 MA10 |
| 回调买入 | 回踩 MA 支撑 | 固定止盈/止损 |
| 趋势持有 | 趋势过滤后突破 |  trailing 或周期卖出 |

具体模板以实现为准；须与 blind 样本特征分布一致。

## 6. 输入与输出

**输入**：

- 盲测会话与操作列表。
- 一致性评估结果（门槛）。
- 标的与周期元数据。

**输出**：

- 候选 DSL 文档。
- 规则摘要（供 H5 展示）。
- `strategy_id` 与版本号。

## 7. 依赖关系

**依赖**：盲测回放、一致性评估、`packages/strategy-dsl`。

**被依赖**：回测验证、执行偏差诊断、策略库、AI 解释。

## 8. 验收标准

- [ ] 仅 blind 样本可生成 ≥1 合法 DSL。
- [ ] 用户未确认前不进入「正式策略」状态。
- [ ] 无 LLM 直接写买卖信号路径（ADR-0003）。
