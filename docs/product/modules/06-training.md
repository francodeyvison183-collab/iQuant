# 06 盲测训练模块

版本：V0.2（方案 A）

## 1. 模块定位

盲测训练模块根据**一致性评估**或**执行偏差诊断**结果，为用户安排专项盲测，帮助其**收敛为一致的交易规则**或**缩小与已确认 DSL 的执行差距**。

## 2. 用户价值

- 一致性低：定向练习（如「只练突破入场 5 轮」）。
- 执行偏差高：针对过早止盈、假突破等场景加练。
- 产品从「一次性报告」变为持续训练工具。

## 3. 核心边界

负责：

- 根据一致性分项或偏差类型生成训练任务。
- 推荐标的/区间/场景（从行情池筛选，非荐股）。
- 训练前后评分对比。

不负责：

- 生成行为策略 DSL（前置由策略生成模块完成）。
- 回测计算。

## 4. 训练类型（示例）

| 触发 | 训练目标 |
| --- | --- |
| 入场不一致 | 固定一种入场逻辑多轮 blind |
| 过早止盈 | 趋势延续行情专项 |
| 跳过信号 | 信号出现时必须决策的练习 |
| 频繁交易 | 空仓等待练习 |

## 5. 依赖关系

**依赖**：盲测回放、一致性/诊断报告。

**被依赖**：工作台待办、AI 训练建议文案。

## 6. 验收标准

- [ ] 一致性不足时可生成 ≥1 个可执行训练任务。
- [ ] 完成训练后可对比前后一致性或执行评分。
